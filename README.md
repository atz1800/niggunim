# יומן הניגונים שלי 🎵

אפליקציה לתיעוד ניגונים — אקורדים, הקלטות וסיפורים.

🔗 **[atz1800.github.io/niggunim](https://atz1800.github.io/niggunim)**

## טכנולוגיות

- React + Vite
- Firebase (Auth, Firestore)
- Google Drive API (אודיו)
- GitHub Pages

## פיתוח מקומי

```bash
npm install
npm run dev
```

נדרש קובץ `.env` עם משתני Firebase (ראה `.env.example`).

## פריסה

נעשית אוטומטית דרך GitHub Actions בכל push ל-`main`.
